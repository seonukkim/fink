# Review Rubric

This rubric is immutable during ordinary task runs.

Claude and machine gates review for:

- objective correctness against the selected task acceptance criteria;
- allowed-path scope;
- tests proportional to risk;
- authority-source boundaries and A0-A2 scoring eligibility;
- financial formula correctness and no invented numbers;
- legal-language boundaries and not-legal-advice framing;
- privacy, copyright, and private-material exclusion;
- Korean-canonical bilingual behavior;
- UI responsiveness and disclaimers when UI is relevant;
- evaluation honesty and frozen-label protection;
- paper claim/result/figure ledger consistency.
