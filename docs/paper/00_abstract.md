# Abstract

FInk is documented as a local-first financial review-priority system for
creator contracts. The paper should present the system as a way to prioritize
financially consequential contract questions for review, not as a legal,
validity, fraud, unfairness, or guaranteed-loss verdict.

The current paper notes use the course-section mapping in `CLM-S7-MAP-PROBLEM`,
`CLM-S7-MAP-METHOD`, `CLM-S7-MAP-DATA`, `CLM-S7-MAP-RESULTS`, and
`CLM-S7-MAP-DISCUSSION`. Measured result highlights are intentionally deferred
to `06_results.md`, where each value cites a `CLAIM_LEDGER.csv` row backed by a
measured `RESULT_LEDGER.csv` row.
