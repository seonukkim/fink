# archive/

Build-process material from the AI-assisted development of FInk, kept for the
record but not part of the shipped product.

- `loop/`, `scripts/agent_loop/`, `scripts/model_research/`, `scripts/invariant_suite.py`,
  `AGENTS.md`, `LOOP.md`, `CLAUDE.md` — the Codex–Claude spec-driven build loop
  and its review charter.
- `docs/specs/`, `docs/FINK_MASTER_SPEC.md`, and the related content guides — the
  original detailed specifications.
- `prompts/` — build and audit prompts.
- `tests/` — tests for the build harness and gates.

The shipped product lives in `src/fink/`, `site/`, and `paper/`.
