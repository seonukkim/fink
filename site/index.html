Overview

Method — four separate dimensions

Outputs are reported side by side and never merged into one total-loss number.

Review Priority

An ordinal signal for which clauses deserve closer review.

Monetary Exposure

Low / base / high scenarios from values or user assumptions.

Time Exposure

Payment timing, recoupment, term, and delay as typed fields.

Evidence & OCR Confidence

Confidence stays separate, so unclear data raises uncertainty.
